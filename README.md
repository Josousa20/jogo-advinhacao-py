# 🎮 Jogo de Adivinhação em Python

Este é um projeto simplese desenvolvido para praticar conceitos fundamentais de Python, como estruturas de repetição, condicionais e manipulação de entrada de dados.

## 🚀 Funcionalidades
- O computador gera um número aleatório entre 1 e 10.
- O jogador tem tentativas ilimitadas para acertar.
- O jogo fornece feedback se o palpite foi alto ou baixo.
- Contador de tentativas para medir o desempenho.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- Biblioteca `random` (para geração do número secreto)
